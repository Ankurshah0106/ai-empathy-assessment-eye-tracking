# AI System to Assess Empathy from Eye-Tracking Data

This repository contains the EyeT eye-tracking datasets and preprocessing notebook used for empathy-focused analysis workflows.

## Contents

- `Pre-Processing.ipynb` - Data cleaning/preparation notebook
- `columns_explained.pdf` - Field descriptions
- `coordinate_system.pdf` - Eye-tracking coordinate system documentation
- `EyeT_group_dataset_*.csv` - Raw eye-tracking trial files

## Data Scope

The dataset includes multiple participants, trials, and experiment groups from the EyeT collection.

## How to Use

1. Open `Pre-Processing.ipynb` in Jupyter Notebook or Google Colab.
2. Set data paths to local CSV files in this repository.
3. Run cells to preprocess and prepare features for modeling.

## Notes

- Large CSV files are included directly in this repository.
- Processing time depends on your system memory and CPU.
